## 🏥 Gestion hospitalière

Ce projet consiste en la conception d’une base de données hospitalière basée sur un schéma relationnel. Il modélise les principales entités du système telles que les patients, les services et les intervenants.

L’objectif est d’assurer une gestion efficace des informations médicales tout en garantissant la cohérence et l’intégrité des données grâce à l’utilisation de requêtes SQL.

Ce projet illustre les principes de modélisation de bases de données et l’exploitation de SQL pour répondre à des besoins métier concrets dans le domaine de la santé.
